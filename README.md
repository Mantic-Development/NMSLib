# NMSLib
Simple to use and basic library for NMS.


Repository:
```xml 
<repository>
    <id>jitpack.io</id>
    <url>https://jitpack.io</url>
</repository> 
  ```

Dependency:
```xml
<dependency>
    <groupId>com.github.Fullpage12</groupId>
    <artifactId>NMSLib</artifactId>
    <version>1.0</version>
    <scope>provided</scope>
</dependency>
  ```
